execve()
Fungsi: Menjalankan program baru. Pada awalnya, ls dijalankan dengan memanggil execve() yang memuat file executable ls ke memori dan memulai eksekusinya.

brk()
Fungsi: Mengatur batas akhir segmen data proses di memori (heap). Digunakan untuk mengalokasikan memori dinamis.

access()
Fungsi: Mengecek apakah program memiliki izin untuk mengakses sebuah file atau direktori (misalnya, memeriksa apakah file executable dapat dibaca/dieksekusi).

openat()
Fungsi: Membuka file atau direktori, di sini biasanya untuk membuka direktori kerja saat ini agar bisa dibaca isinya.

fstat()
Fungsi: Mendapatkan informasi status file (seperti ukuran, tipe, izin) dari file descriptor yang sudah dibuka.

mmap()
Fungsi: Memetakan file atau memori ke dalam ruang alamat proses, biasanya untuk membaca file executable atau data dengan cara efisien.

close()
Fungsi: Menutup file descriptor yang sudah tidak dibutuhkan lagi.

getdents()
Fungsi: Membaca entri-entri direktori (seperti daftar file dalam folder).

write()
Fungsi: Menulis data ke output (biasanya ke layar terminal).

exit_group()
Fungsi: Mengakhiri proses ls secara keseluruhan setelah tugas selesai.
